# Foundations in Bioinformatics

**Building the essential skills for computational biology**

---

## Course Overview

This course provides foundational knowledge in bioinformatics and computational biology, teaching you how biological molecules and processes are abstracted into data that can be analyzed computationally.

### Learning Objectives

By completing this course, you will be able to:

1. **Understand Biological Data** - Comprehend how biological reality becomes digital data
2. **Program for Bioinformatics** - Write Python programs for biological data analysis
3. **Access Biological Databases** - Query and retrieve data from biological databases programmatically
4. **Perform Sequence Alignments** - Execute and interpret sequence alignment algorithms
5. **Apply Statistical Methods** - Use appropriate statistical approaches for genomic data

---

## Course Modules

### [Module 1: Biological Data](module-1.md)
**From Molecules to Information**

Understanding the abstraction process: how physical molecules become digital information

- Types of biological data (Sequence, structure, expression, interaction)
- Measurement and representation
- Abstraction and information loss
- Data quality assessment

**Duration**: 2-3 weeks | **Prerequisites**: Basic molecular biology

---

### [Module 2: Programming Fundamentals](module-2.md)
**Python for Bioinformatics**

Essential programming skills for biological data manipulation

- Python fundamentals (data structures, control flow, functions)
- BioPython library for sequence analysis
- Parsing biological file formats (FASTA, FASTQ, GenBank)
- Writing reusable analysis scripts

**Duration**: 3 weeks | **Prerequisites**: Module 1

---

### [Module 3: Biological Databases](module-3.md)
**Programmatic Data Access**

Accessing and integrating data from biological repositories

- NCBI databases (GenBank, PubMed, GEO, SRA)
- Entrez system for programmatic queries
- UniProt and PDB databases
- Automated batch data retrieval
- Data integration strategies

**Duration**: 2 weeks | **Prerequisites**: Module 2

---

### [Module 4: Sequence Alignment](module-4.md)
**Understanding Alignment Algorithms**

Principles and implementation of sequence comparison methods

- Dynamic programming algorithms
- Global alignment (Needleman-Wunsch)
- Local alignment (Smith-Waterman)
- BLAST and heuristic methods
- Scoring matrices and gap penalties
- Multiple sequence alignment

**Duration**: 3 weeks | **Prerequisites**: Module 3

---

### [Module 5: Statistics](module-5.md)
**Statistical Methods for Bioinformatics**

Appropriate statistical approaches for genomic data analysis

- Descriptive statistics and visualization
- Probability distributions
- Hypothesis testing (t-tests, ANOVA, chi-square)
- Multiple testing correction (Bonferroni, FDR)
- Statistical power and effect sizes
- RNA-seq differential expression analysis

**Duration**: 3 weeks | **Prerequisites**: Module 4

---

## For Students

### Prerequisites

- Basic molecular biology knowledge (DNA, RNA, proteins, central dogma)
- Basic computer skills (file systems, text editors)
- No programming experience required (taught in Module 2)

### How to Use This Course

1. **Read modules sequentially** - Each builds on previous knowledge
2. **Complete exercises** - Hands-on practice is essential
3. **Review examples** - Study code and explanations carefully
4. **Practice with real data** - Apply skills to actual biological datasets
5. **Ask questions** - Use discussion forums or office hours

### Time Commitment

- **Full course**: 14 weeks (3-5 hours per week)
- **Per module**: 2-3 weeks
- **Self-paced**: Work at your own speed

## Course Philosophy

This course emphasizes:

- **Conceptual understanding before programming** - Understand what data represents before manipulating it
- **Hands-on practice** - Learn by doing, not just reading
- **Real-world applications** - Work with actual biological data from publications
- **Reproducible workflows** - Learn best practices from the start
- **Critical thinking** - Question assumptions and evaluate data quality
